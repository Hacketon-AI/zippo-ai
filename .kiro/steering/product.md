# Product Steering

This product is a local-first personal AI assistant for Agung.

Main priorities:
- Runs on Hostinger VPS KVM 2.
- Uses Docker.
- Uses Ollama with Qwen3 as the local model.
- Uses PostgreSQL for cache/history.
- Uses Qdrant for semantic memory/RAG.
- Reduces external API usage.
- Helps with coding, debugging, documentation, and project notes.
